The NIBRS Pre-Certification Tool provides law enforcement agencies responsible for submitting incident data to state Incident-Based Reporting (IBR) programs the ability to test their submissions before formally submitting data to their state IBR programs. State IBR programs can also use the Pre-Certification tool to test their submissions before beginning the NIBRS certification process with the FBI.

The Pre-Cert tool exercises all of the NIBRS data edit and validation rules and provides formatted output of errors so submitters can correct their data submissions and, ultimately, speed the certification process. The Pre-Cert tool is available for free under an open source license that allows the source code to be used with or incorporated into other products. It may be modified to meet state-specific reporting requirements. Source code for the tool is available at https://github.com/SEARCH-NCJIS/nibrs. As an open source project, modifications and updates to the code are encouraged, but not required by the license.
