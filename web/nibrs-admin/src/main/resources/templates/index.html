Welcome to the NIBRS Services Toolset

The NIBRS Services Toolset provides law enforcement agencies, state Uniform Crime Reporting (UCR) programs, and other users with two important capabilities to assist users with processing and understanding NIBRS data:
  1. NIBRS Pre-Certification Tool (PCT)
  2. NIBRS to Summary Reporting Tool (N2S)

NIBRS Pre-Certification Tool (PCT)
The NIBRS Pre-Certification Tool (PCT) can be used by state and local law enforcement agencies to “pre-check” their NIBRS records to identify potential data submission or validation errors. The tool accepts NIBRS data in either the legacy flat file format or XML format.

The PCT tool processes records interactively applying the FBI’s 2019.1 NIBRS validation rules and produces a list of any errors following the standard FBI error file format. The errors can be downloaded for resolution in a variety of formats (Excel, CSV or PDF).

Once records are processed, they are deleted and not retained in any fashion unless they are explicitly saved by the user to produce NIBRS to Summary reports. There are limitations to the tool which are explained on the PCT/Save for Summary Reports page.

NIBRS to Summary Reporting Tool (N2S)
The NIBRS to Summary Reporting Tool (N2S) processes NIBRS data to produce eight (8) standard Summary UCR Reports based on NIBRS data submissions. This enables users to compare crime counts based on NIBRS data to prior crime counts reported under the Summary Reporting System and maintain established trend and time series analyses. Reports are produced interactively, and the user has several options for filtering report contents. More information about the summary reporting capabilities is provided on the N2S/Summary Reports page.

Create an Account The NIBRS Services Toolset is free to use, but users are required to create an account in order to access the tools. Registration ensures that only authorized users have access to their NIBRS data, and enables SEARCH to contact users when updates, enhancements, and other changes are made. Account information will not be shared with any third parties, nor will we use registration information for marketing purposes.
