Welcome to the NIBRS Services Toolset

The NIBRS Services Toolset provides law enforcement agencies, state Uniform Crime Reporting (UCR) programs, and other users with two important capabilities to assist users with processing and understanding NIBRS data:

  1. NIBRS Pre-Certification Tool (PCT)
  2. NIBRS to Summary Reporting Tool (N2S)

NIBRS Pre-Certification Tool (PCT)
The NIBRS Pre-Certification Tool (PCT) can be used by state and local law enforcement agencies to “pre-check” their NIBRS records to identify potential data submission or validation errors. The tool accepts NIBRS data in either the legacy flat file format or XML format. The tool processes NIBRS records interactively using the FBI’s 2019.1 NIBRS rules and provides a list of errors following the standard FBI error file format, which can be downloaded for resolution in a variety of formats (Excel, CSV or PDF). Once records are processed, they are deleted and not retained in any fashion. There are limitations to the tool which are explained on the PCT/Save for Summary Reports page.

NIBRS to Summary Reporting Tool (N2S)
The NIBRS to Summary Reporting Tool (N2S) converts NIBRS data to Summary UCR data, and enables users to produce 8 standard Summary Reports from NIBRS data submissions. This conversion of NIBRS data to Summary Reports enables users to compare NIBRS crime counts to what would have been reported under the rules for Summary UCR reporting, and provides users with crime counts to support legacy Summary Report time series analyses. Reports are produced interactively and the user has several options for filtering report contents. More information about the summary reporting capabilities provided on the N2S/Summary Reports page.
